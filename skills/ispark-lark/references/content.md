# Content Resources

Route:

- docx/wiki document body, document media: docs
- cloud files, folders, import/export, permission, comments: drive
- online spreadsheet cells, formulas, charts: sheets
- bitable/base records, fields, views, dashboards, forms: base
- wiki spaces, nodes, members: wiki
- presentations: slides
- native markdown files: markdown

Read or inspect the resource first when the URL only tells you the container, not the underlying type.
