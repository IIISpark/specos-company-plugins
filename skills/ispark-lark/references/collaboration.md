# Collaboration Resources

Route:

- messages, chats, reactions, chat files: IM
- user lookup and open_id: contact
- events, rooms, free/busy: calendar
- task lists and task records: task
- ended meeting records and transcripts: VC
- minutes upload/download/search: minutes
- known note_id details: note
- mail compose/read/send/drafts/templates: mail

For user-facing sends, drafts, meeting changes, or task assignment, state the target before acting.
